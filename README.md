# JobType PJS With MVVM-HILT-Coroutine



created with: 
- MVVM pattern 
- Live Data 
- Coroutine
- Retrofit 
- Dagger Hilt
- Data Binding
- Navigation Component
- Work Manager
- Material Design 
- minifyEnabled

<b>Note</b>: turn off the window scale, transition scale, and animator duration scale when testing

Feel free to see and contribute
